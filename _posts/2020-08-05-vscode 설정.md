최근까지 python editor로 spyder를 계속 사용하다가 vscode 감성과 conda와의 연동성에 반해 갈아탔다. vscode는 거의 모든 컴퓨터 언어를 다루다보니 python extension을 설치하고 최적화를 위한 extension도 필요하다. 추가적으로 알아가는 사용법이나 extension들을 업데이트해 나갈 계획이다.
 
## <span style="color:#00ADB5;"> python extension</span>

먼저 python extension을 설치해야한다.

## <span style="color:#00ADB5;"> python indent extension</span>

vscode에서는 코드를 작성하다보면 들여쓰기가 안맞게 된다. 이럴 경우에 필요한 것이 python indent extension이다. python 문법에 맞는 indentation을 자동으로 맞추어주어서 너무나 편리하다.
